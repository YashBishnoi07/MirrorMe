# Claude Design System — MirrorMe

## Color Palette
- **Parchment (BG)**: `#f5f4ed` — Warm, editorial feel.
- **Terracotta (Accent)**: `#c96442` — Used for user bubbles, primary CTA.
- **Near Black (Text)**: `#141413` — Almost black, used for body text.
- **Ivory (Surface)**: `#faf9f5` — Used for AI cards and secondary backgrounds.
- **Sage (Secondary)**: `#7d8c7c` — Subdued accent for status indicators.

## Typography
- **Headlines**: 'Lora', Georgia, serif. 
- **Body**: 'Inter', system-ui, sans-serif.
- **Scale**: 
    - Display: 48px / 1.1
    - H1: 32px / 1.2
    - H2: 24px / 1.3
    - Body: 16px / 1.5
    - Small: 14px / 1.4

## Spacing & Layout
- **Base Unit**: 4px
- **Container**: Max-width 1200px
- **Padding**: 24px-32px for main sections

## Components
- **Chat Bubbles**:
    - User: Terracotta background, ivory text, 16px radius, bottom-right sharp corner.
    - Mirror: Ivory background, dark text, subtle border, serif quotes.
- **Input Bar**: Fixed bottom, floating appearance, huge padding, minimal border.
- **Cards**: Minimal border `#e8e6db`, whisper shadows.
