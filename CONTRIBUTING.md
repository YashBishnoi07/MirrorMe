# Contributing to MirrorMe

First off, thank you for considering contributing to MirrorMe! It's people like you that make MirrorMe such a great tool.

## How Can I Contribute?

### Reporting Bugs
- Use the GitHub issue tracker to report bugs.
- Describe the bug in detail and provide steps to reproduce it.

### Suggesting Enhancements
- If you have an idea for a new feature, please open an issue and explain why it would be useful.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. Follow the existing code style.
3. Ensure the backend and frontend both start without errors.
4. Issue a pull request!

## Local Development
- MirrorMe relies on local LLMs. Ensure you have **Ollama** running.
- Follow the setup guide in the [README.md](README.md) to get your environment ready.

## Code of Conduct
By participating in this project, you agree to abide by the terms of the Contributor Covenant.
